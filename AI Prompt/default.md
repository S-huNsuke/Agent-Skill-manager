# 全局规则

## 语言与沟通
始终使用中文与用户对话

## 设备环境
用户设备：MacBook (macOS)

## 技术栈

### 编程语言
编程语言：Python、Go、TypeScript，根据需求选择最合适的语言

### 包管理工具
Python：使用 uv 管理依赖和虚拟环境
TypeScript/Node：使用 pnpm 管理依赖
使用 mise 管理各种语言、工具的版本

### Python 虚拟环境
运行 Python 命令时均使用项目根目录的 .venv 虚拟环境
激活虚拟环境后再执行 Python 脚本或包管理命令

## 代码规范
所有函数必须添加函数级注释，说明功能、参数和返回值

## 工具使用约束
Glob 不搜索隐藏目录（以 . 开头），查看 ~/.claude/ 等隐藏路径下的内容时，必须用 Bash ls，不能用 Glob

## 决策与选择
在进行选择时需要多询问用户的意见，不要单方面决定，给出选项并让用户选择最适合的方案

## 项目实施规范
时刻遵循项目计划文档及其他规范文件
在项目实施过程中，及时调整计划，解决出现的问题
